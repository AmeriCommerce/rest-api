gift_certificates
=================

```shell
GET /api/v1/gift_certificates
```

**Required scope**: `read_marketing`, `marketing`

Nested Resources
----------------

Accessible via the `?expand=` parameter or `/api/v1/gift_certificates/{id}/{nested_resource}`.

GET /api/v1/gift_certificates?expand=transactions
